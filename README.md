# 👁️ VisionTest - 智能视力检查应用 v1.2.0

由福宝 (Fubao) 🐼 开发的自动视力检测应用

## ✨ 新版本功能（v1.2.0）

### 🎯 **核心改进**

#### 1. 📏 **摄像头距离自适应**
- **实时检测**用户眼睛到屏幕的距离
- **自动调整**视标大小以匹配实际观看距离
- **智能提示**当距离不适当时给出指引

**工作原理：**
```
摄像头 → 检测手部大小 → 计算观看距离 → 调整 E 字大小
太近 (<40cm)    ↓          调小视标 0.7x
合适 (40-60cm)   ↓          标准大小 1.0x  
太远 (>70cm)     ↓          调大视标 1.3x
```

#### 2. ✋ **挥手表示"没看到"**
替代之前的握拳/静止操作，更符合直觉：
- **🖐️ 左右摆动手掌** → 像打招呼一样摇手
- ✅ 系统自动识别挥手动作
- ✅ 记录为"没看到该视标"但不进入下一行
- ✅ 用户可以重新尝试这一行

#### 3. 🎮 **滑动手势判断方向**
延续 v1.1 的优化：
- ⬆️ 向上滑 → E 口朝上
- ⬇️ 向下滑 → E 口朝下
- ⬅️ 向左滑 → E 口朝左
- ➡️ 向右滑 → E 口朝右

## 📱 使用流程

### 准备阶段
1. 打开应用并点击「新测试」
2. 确保手机摄像头能拍摄到您的手部区域
3. 将手机固定在距离眼睛约 50cm 的位置

### 测试过程

#### 情况 A：看到了 E 字开口方向
```
1. 看清 E 字的朝向
2. 手指在屏幕上向对应方向滑动
3. 系统自动记录结果并进入下一行
```

#### 情况 B：没看到或看不清 E 字
```
1. 左右摆动手掌（像跟别人打招呼）
2. 系统检测到挥手动作
3. 标记为"没看到"，停留在当前行
4. 可以调整距离后重试
```

### 查看结果
- 底部显示"视力总结"卡片
- 最佳和最差视力值
- 最近 20 条测试记录及对应手势类型

## 📊 距离估算原理

### 基于视觉大小的透视公式

```
真实距离 = (物体真实尺寸 × 焦距) / 图像中像素尺寸
```

**简化实现：**
```kotlin
// 假设成年人的手掌宽度约为 8-10cm
val referenceHandWidth = 90f // mm

// 从摄像头图像中测量手掌像素宽度
val pixelWidth = detectHandPixelWidth()

// 基于像素大小估算距离级别
if (pixelWidth > ref * 1.5) {
    return TOO_CLOSE   // 手掌很大 → 距离很近
} else if (pixelWidth < ref * 0.5) {
    return TOO_FAR     // 手掌很小 → 距离很远
} else {
    return OPTIMAL     // 正常范围
}
```

### 缩放系数计算

```kotlin
fun getScaleFactorForDistance(): Float {
    when (distanceEstimate) {
        TOO_CLOSE -> 0.7f   // 调小视标，避免过大
        OPTIMAL -> 1.0f     // 标准大小
        TOO_FAR -> 1.3f     // 调大视标，避免过小
    }
}
```

## 🎬 挥手识别机制

### 挥手动作特征

1. **水平主导运动**
   - X 轴位移 > Y 轴位移 × 2
   - 总位移超过阈值（如 50 像素）

2. **方向变化**
   - 从左到右或从右到左
   - 至少一次方向切换才算完整挥手

3. **速度限制**
   - 过快可能误判为滑动
   - 过慢可能无法触发
   - 合理范围：200-800 像素/秒

### 检测流程图

```
开始 → 监测手部移动 → 是否主要是水平移动？
                              ↓是
                     是否超过最小位移阈值？
                              ↓是
                     是否有方向切换？
                              ↓是
                    ✅ 判定为挥手动作
                              ↓
                    📝 记录"没看到"结果
```

## ⚠️ 重要提示

### 距离检测注意事项
1. **光线充足**：确保摄像头能清晰看到手部
2. **背景简单**：纯色背景更容易检测
3. **手掌可见**：不要遮挡手部
4. **稳定手持**：减少剧烈晃动影响精度

### 挥手操作建议
1. **幅度适中**：不要太快也不要太慢
2. **幅度足够**：左右各移动至少 5cm
3. **持续稳定**：保持 0.3 秒以上的摆动感
4. **单侧开始**：可以从一边先开始挥动

### 设备要求
- Android 7.0+ (API 24+)
- 后置摄像头（用于检测）
- 推荐屏幕尺寸：5.5 英寸以上

## 📊 视力读数参考

| 数值 | Snellen | 含义 | 建议 |
|------|---------|------|------|
| ≥1.0 | 20/20 | 正常视力 | ✅ 继续保持 |
| 0.8-0.9 | 20/25 | 轻度下降 | 💡 注意用眼卫生 |
| 0.6-0.7 | 20/30 | 中度下降 | 🩺 建议眼科检查 |
| 0.3-0.5 | 20/40-20/60 | 明显下降 | 🚨 尽快就医 |
| <0.3 | 20/80 以下 | 严重下降 | 🏥 立即就医 |
